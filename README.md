
**Employee Attrition Analysis | HR Analytics Dashboard**

1. Project Overview

This project analyzes employee attrition trends to help the HR department understand why employees are leaving and which factors influence attrition.

2. The dashboard provides insights into:

- Overall attrition rate
- Age distribution
- Salary impact
- Years at company
- Job roles with highest attrition
- Education background impact
- The objective is to support data-driven HR decision-making and improve employee retention strategies.

3. Tools & Technologies Used

- Power BI
- DAX (for calculated measures)
- Data Modeling
- Data Cleaning & Transformation
